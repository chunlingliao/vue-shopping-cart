```

```

### DEMO
```
https://chunlingliao.github.io/vue-shopping-cart/
```

### 簡介
```
此作品為 Vue Cli 前端練習，其主要功能有以下:

∙ 登入功能
∙ 選擇產品加入購物車
∙ 點擊產品後可選擇數量及加入功能
∙ 添加、刪除、計算總金額
```
